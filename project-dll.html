IC Design · EE124 · 2026

8 GHz Delay
Lock Loop

Cadence Virtuoso Spectre 45nm CMOS Maestro Analog IC Design

Overview

Designed a low-power, low-jitter 8 GHz Delay Lock Loop (DLL) in 45nm CMOS technology as part of EE124 (Mixed-mode Integrated Circuits). The DLL is used to synchronize a clock signal with minimal phase offset and jitter — critical in high-speed communication and processor clock distribution networks.

Architecture

The DLL consists of four main subsystems, each of which I designed and verified individually before integrating:

  • Phase-Frequency Detector (PFD) — detects phase and frequency difference between the reference clock and the feedback signal. Implemented with a pair of D flip-flops and a reset path to generate UP/DOWN correction pulses.
  • Charge Pump & Loop Filter (CP/LF) — converts PFD output pulses into a smooth control voltage. The loop filter sets the loop bandwidth and stability characteristics of the DLL.
  • Voltage-Controlled Delay Line (VCDL) — a chain of delay cells whose delay is tuned by the control voltage from the loop filter. The total delay of the VCDL is driven to exactly one reference clock period.
  • Feedback path — connects the VCDL output back to the PFD, closing the loop.

Design Specifications

Technology45nm CMOS
Target Frequency8 GHz
Peak-to-Peak Jitter< 11 ps (clean supply), verified with noisy supply
Power Consumption< 3.8 mW
Lock RangeVerified across clean and noisy clock inputs
Simulation ToolCadence Spectre via Maestro

Simulation & Verification

Simulations were run in Cadence Maestro using Spectre. I verified lock behavior under both clean and noisy supply and input clock conditions. The DLL achieved lock reliably in both scenarios, with peak-to-peak jitter staying under 11 ps and total power consumption under 3.8 mW.

Parametric sweeps were used to characterize the VCDL delay vs. control voltage, and transient simulations confirmed lock acquisition time and steady-state jitter.

Outcome

This project was awarded Best DLL Project in EE124 for having the best combined performance across power consumption, phase offset, and peak-to-peak jitter among the class submissions.
